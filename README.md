# Belajar Open API V3.0.3

## Todolist API
- `GET` /todolist
  - Get all todolist
- `POST` /todolist
  - Create new todolist
- `PUT` /todolist/{todolistId}
  - Update existing todolist
- `DELETE` /todolist/{todolistId}
  - Delete existing todolist
